Laser Art. Elektro Serwis — Wersja z galerią i obrazkami (GitHub Pages)

Jak podmienić obrazy:
1) Wejdź do repo → folder images/
2) Prześlij swoje pliki JPG/PNG i nadaj im te same nazwy, żeby zastąpić placeholdery:
   - hero.svg (główne zdjęcie w nagłówku)
   - praca1.svg, praca2.svg, praca3.svg, praca4.svg, praca5.svg
3) Po commicie odśwież stronę — zobaczysz własne zdjęcia.

Wdrożenie na GitHub Pages (skrót):
- Add file → Upload files → wgraj cały folder images/ oraz index.html
- Settings → Pages → Deploy from a branch → main / (root)
- Odczekaj chwilę i otwórz link do strony.
