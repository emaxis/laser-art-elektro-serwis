Instrukcja — Laser Art. Elektro Serwis (GitHub Pages z przykładowymi plikami JPG)

1. W repozytorium na GitHub wrzuć plik `index.html` i folder `images/`.

2. W folderze `images/` są pliki JPG z nazwami:
   - hero.jpg  (główne zdjęcie w nagłówku)
   - praca1.jpg … praca5.jpg (galeria)

   ❗ Obecne pliki to tylko placeholdery (puste).

3. Aby podmienić na swoje:
   - przygotuj własne zdjęcia JPG,
   - nazwij je tak samo (hero.jpg, praca1.jpg …),
   - wrzuć do folderu `images/` → zastąpią placeholdery.

4. Włącz GitHub Pages:
   - Repozytorium → Settings → Pages
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Save

5. Odśwież stronę — Twoje zdjęcia pojawią się na stronie.
